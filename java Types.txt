Compresion files of Java technology
-----------------------------------------
1. jar (java archive)  -> compressed file format having only .class files
2. war (web archive) -> compressed file format having web related files like html/java ee
			class/ css / images /audio / video etc
3. ear (enterprise archive) -> compresed file format for enterprise application develipment
 			 apps like ejb or any distruted application development
